HTML Questions

HTML stands for HyperText Markup Language and used to build static webpages.

HTML stands for HyperText Markup Language and used to build static webpages. CSS stands for Cascading Style Sheets and used to add style to the webpage. JavaScript is a programming language that is used for adding functionalities to the HTML webpages.

doctype describes the version of html that we are using.

tags are part of an element, and element is the combination of tags and content

HTML entities are character sequences that represent special characters that are not part of a standard keyboard

HTML 5 is an upgraded version of HTML 4, offering new features such as audio, video, and canvas tags for drawing graphics.Both HTML 4 and HTML 5 are compatible with many different web browsers.

There are mainly Two types of elements in html and they are Inline and Block elements. Inline Elements occupies the width of the element and next element starts in the same line, but Block Elements occupies the full width of elements and next element starts in the next line

We can make the square box of red color in html without any content by taking div tag and giving the height, width, red backgroundcolor.

Attribute provides the extra information about information of an content in the tag. Some of the attributes that i know are src, href, alt, title, id, class, style, etc.

head tag contains link tag, title tag, Script tag, and meta tag

Meta tags are used in the head tag of HTML documents to provide information about a website's content to search engines and visitors

Search engine optimization (SEO) is the process of optimizing a website to rank higher in search engine results. SEO is important because it helps users find a website and decide whether to visit it, which can lead to more traffic and more business.

An iframe is an HTML element that allows you to embed another HTML document within your own. It's a powerful feature that lets you include content from other websites on your own site

A semantic element clearly describes its meaning to both the browser and the developer. but non-semantic elements are used for mainly styling purpose only. we are using semantic elements rather than non-semantic elements to enhance the quality of the code and SEO purpose.

relative path is the path that we given from the current working directory. Absolute path is the exact path of the fileaddress

Git Questions

Git is a free, open-source version control system (VCS) that's used by developers to track changes to their code.

GitHub is a cloud-based platform that allows users to share and collaborate on code

We can Initialize the git in our local by using the command "git init" in terminal

working directory --> staging area --> local area --> remote area

git add

git commit -m "message"

git remote add origin <-link->

n version control systems, a repository is a data structure that stores metadata for a set of files or directory structure.

There is only one way to push the code from local to remote by using the command "git push origin main"

we can create the branch by using command "git branch <-branch name->

by using the command "git branch"

by using the command "git checkout <-branch-name-> "

by using the command "git log --oneline"

git status command describes the status of the local repository.

there are 3 ways to get the code from remote to local. they are "git fetch", "git pull", "git clone".

A "fork" on GitHub is essentially a personal copy of an existing repository, allowing you to make changes to the code without affecting the original project; to fork a repository, simply navigate to the desired repository on GitHub and click the "Fork" button, which will create a new copy of the repository under your own account.

The main difference between git fetch and git pull is that git fetch downloads changes to the local Git repository, while git pull downloads and merges those changes into the local repository.

In Git, git stash saves changes in a separate location, while git diff compares and shows changes in a project.

In Git, git reset and git remote are commands that can be used to reset a local branch to a remote branch or to reset a remote branch to origin.

Syntax of Css:
selector { property:value;}

there are 3 ways of attaching the css to html. they are inline stling, internal styling, External Styling.

some of the selectors that we have are simple Selectors, combinator selectors, Pseudo-class selectors, Pseudo-element selectors and variable selectors.

display:block converts elements into block elements, Display:inline converts elements into inline elements, and Display:inline-block converts elements into inline elements but we can add borders and margins to it.

pseudo-classes target an element's state, while pseudo-elements target specific parts of an element

The Box model contains Padding, margin, and borders

Padding is the value of space between content and border. Margin is the value of space outside the border.
